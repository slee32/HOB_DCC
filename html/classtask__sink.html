Detailed Description

This task reads measurements that were taken by the data acquisition task in task_daq.* and puts those measurements into a queue.

void task_sink::run ( void  )
virtual

This run method is called by the RTOS and contains a loop in which the task checks for data and sends it if appropriate.

This method receives data from the data acquisition task in a queue and sends the information or saves it, depending on what its serial device pointer points to.
